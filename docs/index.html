RAII Memory

Automatic native memory via AutoCloseable. No manual free_array() needed.

NumPy API

Npc.zeros(), Npc.sum(), Npc.dot() — familiar names, Java conventions.

Try-with-Resources

try (NpcArray a = Npc.ones(3,4)) { ... } — arrays auto-close, no leaks.

Print & I/O

a.print(), Npc.savetxt(), Npc.loadNpy() — .npy, .npz, CSV.

Zero-Cost Move

Efficient native pointer transfer. Deep copy on copy, O(1) on move.

Linear Algebra

SVD, QR, LU, Cholesky, Eig — all return structured result types.

Installation

Prerequisites

JDK 11+  ·  CMake 3.16+  ·  libnpc (pre-built C library)  ·  libnpcpp (header-only C++ wrapper)  ·  libzip (optional, for .npz I/O)

Build from Source

Terminal
# 1. Build libnpc (one-time)
cd /path/to/libnpc && make lib

# 2. Build JNI native library
cd libnpcJava && mkdir -p build && cd build
cmake .. -DLIBNPC_ROOT=/path/to/libnpc -DLIBNPCPP_ROOT=/path/to/npc++ -DCMAKE_BUILD_TYPE=Release
make -j$(nproc)

# 3. Build Java & run tests
cd .. && mvn compile && mvn test

Quick Start

Main.java
import libnpc.Npc;
import libnpc.NpcArray;

public class Main {
    public static void main(String[] args) {
        // Create arrays — they auto-close via try-with-resources
        try (NpcArray a = Npc.ones(3, 4);
             NpcArray b = Npc.full(new int[]{3, 4}, 2.0)) {

            NpcArray c = Npc.add(a, b);    // element-wise add
            NpcArray d = Npc.sqrt(c);      // element-wise sqrt

            double total = d.sum().scalarDouble();
            System.out.println(total);     // 7.794

            NpcArray I = Npc.eye(3);
            SvdResult svd = Npc.svd(I);
        }
    }
}

NpcArray Class

The NpcArray class is the central type of libnpc Java. It implements AutoCloseable for automatic native memory management.

Lifecycle

MethodDescription
close()Frees native memory. Idempotent — safe to call multiple times.
isClosed()Returns true if already closed.

Metadata

MethodReturnDescription
ndim()intNumber of dimensions.
size()intTotal element count.
shape()int[]Shape array.
shape(int axis)intSize of given axis.
dtype()DTypeElement data type.

Element Access

MethodDescription
getDouble(int... indices)Multi-dim getter.
setDouble(double v, int... indices)Multi-dim setter.
getDouble(int i)1-D getter.
setDouble(int i, double v)1-D setter.
scalarDouble()Get first element as double.
toDoubleArray()Copy all elements to new double[].

Printing & I/O

a.print();     // print values to stdout
a.info();      // print metadata (dtype, ndim, shape)
a.toString();  // → "Array(dtype=10, ndim=2, shape=[3,4])"
a.savetxt("data.csv");
a.saveNpy("data.npy");
